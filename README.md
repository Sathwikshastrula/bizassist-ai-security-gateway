BizAssist AI Security Gateway
Overview

BizAssist AI Security Gateway is an enterprise-grade AI security layer built to protect LLM applications from prompt injection, confidential data extraction, and malicious user interactions.

The project demonstrates a multi-layer AI safety architecture using FastAPI, NVIDIA NeMo Guardrails, and Llama 3.1.

Unlike traditional keyword filtering, BizAssist uses semantic similarity matching with confidence thresholds to distinguish malicious prompts from legitimate business queries.
